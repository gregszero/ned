# Command Center Dashboard

## Context
Build a showcase page at `/pages/command-center` that demonstrates OpenFang's capabilities in one glance — a live dashboard with 7 widgets covering monitoring, email, workflows, and quick actions.

## Files to Create (5 new files, 0 modifications)

### 1. `fang/widgets/system_overview_widget.rb` — System Stats Grid
- 3x3 grid of stat counters: Conversations, Pages, Skills, Scheduled Jobs, Heartbeats, Workflows, Triggers, Notifications, Data Tables
- Each cell shows count + a contextual sub-badge (e.g. "3 pending", "2 active")
- Refreshable every 60s, green header (`#16a34a`)
- Pattern: same as `heartbeat_monitor_widget.rb` (query DB, compare content, update)

### 2. `fang/widgets/gmail_inbox_widget.rb` — Recent Unread Emails
- Shows 5 most recent unread emails (from, subject, snippet, date)
- Three fallback states: "Gmail Not Connected" / "Loading..." / "Inbox Zero"
- Refreshable every 300s, red header (`#ea4335`)
- Gmail `search` returns `{id:, subject:, from:, date:, snippet:, labels:}` (symbol keys)
- Caches email data in `metadata['emails']` to detect changes
- `rescue => e` around Gmail API calls

### 3. `fang/widgets/workflow_monitor_widget.rb` — Workflow Progress
- Shows 5 most recent workflows with name, status badge, and segmented progress bar
- Each step = one bar segment colored by status (green/blue/red/gray)
- Refreshable every 30s, purple header (`#8b5cf6`)
- Uses `includes(:workflow_steps)` to avoid N+1

### 4. `fang/widgets/quick_actions_widget.rb` — Action Button Grid
- 2-column grid of `outline sm` buttons using `data-fang-action` pattern
- Not refreshable — actions configured via metadata
- Default actions: Daily Briefing, Check Gmail, List Heartbeats, Pending Approvals
- Amber header (`#f59e0b`)
- Posts to `/api/actions` with `action_type: 'run_skill'` (supported by `web/app.rb:448`)

- Creates/finds page with slug `command-center`
